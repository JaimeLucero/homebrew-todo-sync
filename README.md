# homebrew-todo-sync

Homebrew tap for installing todo-sync.

## Installation

```bash
brew tap jaimelucero/todo-sync https://github.com/jaimelucero/homebrew-todo-sync
brew install todo-sync
```

## Formula

- **todo-sync**: Bidirectional sync between TODO.md and GitHub Issues

## Development

To use the local tap during development:

```bash
brew tap jaimelucero/todo-sync /path/to/homebrew-todo-sync
brew install todo-sync
```
